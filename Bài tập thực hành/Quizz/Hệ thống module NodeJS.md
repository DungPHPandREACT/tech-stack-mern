Câu 1: Đâu là loại module trong Node.js
Select one or more:
A. Third party module
B. Core module
C. Export module
D. Local modules
The correct answers are: Local modules, Core module, Third party module

Câu 2: Hãy chọn core module trong Node.js
A. http
B. mysql
C. Mongoose
D. ExpressJS
The correct answer is: http

Câu 3: Chọn phát biểu đúng về local module
Select one or more:
A. Local module là những module được tích hợp sẵn khi cài Node.js
B. Local module là những module do chính bạn viết trong dự án của bạn.
C. Local module là những module được cộng đồng phát triển trên NPM
D. fs là một local module
The correct answer is: Local module là những module do chính bạn viết trong dự án của bạn.

Câu 4: Chọn phát biểu đúng về module system.
Select one or more:
A. Trong ES module thì xuất nhập module bằng module.exports và require
B. Module system cho phép chúng ta tổ chức lại mã code, ẩn đi những thông tin không cần thiết.
C. Trong CommonJS thì xuất và nhập module bằng export và import
D. Trong ES module thì xuất nhập module bằng export và import
The correct answers are: Module system cho phép chúng ta tổ chức lại mã code, ẩn đi những thông tin không cần thiết., Trong ES module thì xuất nhập module bằng export và import

D. Bạn có thể cài đặt một module trên NPM bằng cú pháp: ```npm install --save <Tên module>```
The correct answers are: NPM là viết tắt của node package manager , NPM là kho quản lý và lưu trữ những module do cộng đồng phát triển, Bạn có thể cài đặt một module trên NPM bằng cú pháp: ```npm install --save <Tên module>```

Câu 7: Chọn phát biểu đúng về ES module
Select one or more:

B. ES module nhập module bằng require
C. ES module nhập module bằng import

B. Xóa file
C. Theo dõi thông tin về hệ thống
