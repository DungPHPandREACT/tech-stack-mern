Câu 1. Chọn những phát biểu đúng về Web Service
Select one or more:
A. Việc thực hiện truyền nhận thông tin giữa các web service được thực hiện qua giao thức HTTP
B. Web Service cung cấp một phương tiện tiêu chuẩn để tương tác giữa các ứng dụng phần mềm khác nhau, chạy trên nhiều nền tảng và / hoặc khuôn khổ khác nhau
C. Web Service là trung tâm của Kiến trúc Hướng Dịch vụ (SOA)

The correct answers are: Web Service cung cấp một phương tiện tiêu chuẩn để tương tác giữa các ứng dụng phần mềm khác nhau, chạy trên nhiều nền tảng và / hoặc khuôn khổ khác nhau, Web Service là trung tâm của Kiến trúc Hướng Dịch vụ (SOA), Việc thực hiện truyền nhận thông tin giữa các web service được thực hiện qua giao thức HTTP

Câu 3: Chọn những phát biểu đúng về REST API
Select one or more:
A. Được triển khai hoàn toàn dựa trên HTTP
B. Có thể sử dụng định dạng dữ liệu JSON và XML
C. Là một tập hợp các API được thiết kế theo kiến trúc REST
The correct answers are: Là một tập hợp các API được thiết kế theo kiến trúc REST, Được triển khai hoàn toàn dựa trên HTTP, Có thể sử dụng định dạng dữ liệu JSON và XML

Câu 4: 4 phương thức phổ biến thường gặp nhất của RESTful API
A. GET, POST, PUT, PATCH
B. GET, POST, PUT, DELETE
C. GET, PUT, PATCH, DELETE
D. POST, PUT, PATCH, DELETE
The correct answer is: GET, POST, PUT, DELETE

Câu 5: 4 hành động phổ biến nhất khi thiết kế Restful API, đó chính là
Select one or more:
A. R - READ
B. U - UPDATE
C. C - CREATE
D. D - DELETE
E. R - ROLLBACK
The correct answers are: C - CREATE, R - READ, U - UPDATE, D - DELETE

Câu 6: Một Resource (tài nguyên) trong RESTful API có thể là
Select one or more:
A. Một chủ thể đơn lẻ (Singleton)
B. Một tập hợp bao gồm một số tập hợp con (sub-collection)
C. Một tập hợp (Collection)
The correct answers are: Một chủ thể đơn lẻ (Singleton), Một tập hợp (Collection), Một tập hợp bao gồm một số tập hợp con (sub-collection)

Câu 8: Một số quy tắc thường được sử dụng để thiết kế URI
Select one or more:
A. Sử dụng "/" để ngăn cách quan hệ trong resource
B. Sử dụng "_"
C. Sử dụng chữ cái thường
D. Sử dụng "/" ở cuối URI

The correct answers are: Sử dụng "/" để ngăn cách quan hệ trong resource, Sử dụng  "-", Sử dụng chữ cái thường

Câu 9: Các điểm đặc trưng của GraphQL là

A. Sử dụng một type system để khai báo dữ liệu.
B. Cho phép client xác định chính xác những dữ liệu gì họ cần
